Help Box (Bloque de Ayuda)

Defining relative clauses(cláusulas relativas definitorias) cuando usar WHICH y cuando usar THAT

  • Podemos definir a personas o cosas con una cláusula relativa definitoria (restrictiva).
  • Usamos el pronombre relativo "WHO" para referirnos a una persona; también podemos usar "THAT".
  • Un blogger es una persona who/that mantiene un blog o publica un diario en línea.
  • Usamos el pronombre relativo "WHICH" (o "THAT") para referirnos a cosas, no a personas.
  • Esto está integrado en un solo chip WHICH/THAT ejecuta las instrucciones del programa y coordina las actividades que tienen lugar dentro del sistema informático.
  • Los pronombres relativos se pueden omitir cuando son el objeto de la cláusula relativa. La placa de circuito principal (which/that) tienes dentro de tu sistema se llama placa base (motherboard).

Identificación de Sustantivos y Adjetivos

Regla para Identificar Sustantivos

  • Definición: Nombres de personas, lugares, cosas o ideas.
  • Cómo Identificar: Pregunta: "¿Qué es esto?"
  • Ejemplos:
    • Personas: profesor, cliente
    • Lugares: banco, escuela
    • Cosas: computadora, dinero
    • Ideas: información, velocidad

Regla para Identificar Adjetivos

  • Definición: Palabras que describen o califican sustantivos.
  • Cómo Identificar: Pregunta: "¿Cómo es esto?" o "¿Qué tipo de...?"
  • Ejemplos:
    • Describiendo cosas: grande, rápido
    • Describiendo personas: amigable, alto
    • Describiendo lugares: bonito, concurrido
    • En tu oración: financiero, grande, alto
